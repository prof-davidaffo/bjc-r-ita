Contact List App

All cell phones have some kind of Contacts list. In this lab, you will build a Contacts app yourself.

  1. Load this starter version of the app.
  2. Press the "Add Contact" button on the stage, and read the script to understand how it works. Each new contact is a list (item 1 is a name and item 2 should be a phone number), so the list of contacts is a list of lists.
  3. Fix the block so that it works similarly to .
  4. Clear your contact list and add a few made-up people to make sure everything works.
  5. Write a reporter that takes any part of a person's name as input, and reports that person's phone number, or "not found" if the name is not in the contact list. You will need:
    1. , which lets you find out whether one string (a contact's name in the list) contains another string (a name or part of a name).
    2. , which filters the list for only the items that make the function true.
    3. , which returns true if the list is empty and false otherwise.
  6. When someone calls you, the phone company sends only the calling number to your phone, not the person's name. But if the number is in your contact list, your phone figures out the name. So you'll also want a block that searches for a phone number and reports the contact's name.
Make sure everything works the way you want it to before going on. Then save!
