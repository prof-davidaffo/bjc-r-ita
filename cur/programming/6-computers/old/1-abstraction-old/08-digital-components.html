The Digital Domain: Components

On this page you'll explore the parts (components) of the computer hardware.

For as long as people have been teaching other people about computers, they always say "A computer consists of the processor, memory, and input/output (I/O) devices."

In the early days of computers, it was easy to point out the huge racks of equipment that contained each of those three components:

These are pictures of a powerful computer from the 1970s. Each of the big racks full of circuitry is about the size of a refrigerator. Today, a much more powerful computer—more speed, more memory—fits inside a cell phone.

Because today's processors and memory are so small, most of what we see is the I/O devices, like keyboard, mouse, screens.

    2. Scanner: An input device that can optically scan an image, a writing, etc. and convert it to a digital image that can be stored in the computer.

    CPU (Central Processing Unit): The "brain" of the computer, the microprocessor chip where programs are run. It is one of the most expensive parts of the hardware. Its speed is measured in gigahertz (GHz). Modern CPUs can perform multiple tasks simultaneously.

    Image credit: Wikipedia user Solipsist.

    RAM (Random Access Memory): Computer's high-speed, working memory. It temporarily stores data and instructions for programs that run on the computer. Its capacity is measured in gigabytes (GB) and its speed is measured in megahertz (MHz).

    Image credit: Wikipedia user Cyberdex

    Expansion cards: A circuit board that can be inserted to add functionality to a computer system.

    Image credit: Wikipedia user Rackmount-guy.

    Power supply unit: Converts electricity from the wall into the form that the other computer components use.

    Optical Disc Drive: I/O device that reads data from and writes data to CDs and DVDs.

    Hard Drive: I/O device. Long term storage memory of the computer. Mechanical drives use a mechanical arm to read and write data on a rotating disk. "Solid state" drives have no moving parts. Storage capacity is measured in gigabytes (GB).

    Video credit: Wikipedia user Peter17

    Motherboard: A circuit board that holds and connects various components of the computer and allows their communication.

    Image credit: Wikipedia user Julianprescott2604juuly.

    Speaker: I/O device through which the computer outputs sound.

    Monitor: I/O device. Generally, monitors are output devices where the computer visually displays information. Touch-screens combine the functions of output and input. The size of the screen is measured as the diagonal distance between opposite corners.

    Keyboard: An input device on which the user can type and communicate with the computer.

    Mouse: An input device that allows the user to visually interact with graphical objects displayed on the monitor.

    External Hard Drive: I/O device. A supplementary hard drive that typically connects to a computer through the USB port and allows for extra data storage.

    Printer: An output device that gives printed output.

    Image credit: Wikipedia user HereToHelp

    1. Decide how to ask and answer a question (Will the program choose randomly from component list? Will the user click a number on the stage to answer?)
