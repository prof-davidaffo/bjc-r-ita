Optional Details on Adding an Input Variable

  • If you don't see the + marks, uncheck "Plain prototype labels" in the Settings menu.
    The + marks: In the Block Editor, clicking on a word lets you change the word, and clicking on a + sign lets you put a new word or input variable there. Your new input variable will go after the title, so click the + at the end.
    See also Snap! Manual pages 19-20 and, for more advanced info, pages 27-31.

  • Naming the Input Variable: Choose a variable name, like size, that makes its meaning clear. Type that name in the "Create input name" field, ad click "OK".

    You have two choices about how to treat that new text:
    1. Input name is automatically selected for you. That's what you want now, so just leave it selected.
    2. But if you ever want to change a block's title, you could click Title text (gray in this picture) and type the new words. (For example, to change the block name from draw square to draw square picture, click on "Title text" and type the word "picture.")

    Your hat block has changed: there is now a variable named size after the title:

  • To use the size variable, drag the variable where you want to use it. In this case, that's in the move block. If you ever need another copy of a variable, just drag another down.

    If you have extra copies of a block or variable or script that you want to get rid of, you can drag them to the palette at the left of the snap window.

This draw polygon block generalizes the pattern of draw square, draw triangle, draw hexagon, and so on. It does so by adding an input that expresses the difference from one block to another, namely, the number of sides. This is another example of abstraction.
