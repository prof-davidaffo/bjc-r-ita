Fila di case

In questo progetto, userai astrazioni dentro altre astrazioni per costruire una fila di case.

  1. "U1-FilaDiCase"
  2. Crea un blocco disegna quadrato con input dimensione. Assicurati di provarlo con input diversi.
    Hai imparato a creare un blocco con input nel Laboratorio 3 Pagina 3: Blocchi con input.
    Clicca per vedere un modo di disegnare quadrato senza un input.
  3. Usando disegna quadrato come esempio di partenza, crea e prova un blocco disegna triangolo con input dimensione che disegni un triangolo equilatero.
    Hai imparato a fare triangoli nel Laboratorio 3 Pagina 2: Angoli e rotazioni.
  4. Ora usa disegna quadrato e disegna triangolo insieme per creare un blocco disegna casa con input dimensione che disegni la forma di una casa: un quadrato con sopra un tetto triangolare.
    Potresti dover fare debugging per far collegare correttamente le due forme.
