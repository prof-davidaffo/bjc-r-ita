Make It a Game

On this page, you'll make your app more like a game. Alonzo will start to move on his own, making it harder to click him, and each time the player clicks on Alonzo, he will become a little lighter. The goal of the game is to click on Alonzo until he disappears.

Blocks that look like this can be used to detect an event such as clicking on the sprite, pressing a key, or clicking the green flag button (). They tell the script to start when that event occurs. Because these blocks can sit only on top of a script (as a way to start the script), Snap! programmers call them "hat blocks."

Find these blocks in the yellow Control palette, and see what other choices are in their drop-down menus.

Don't think that every script needs a hat block. You can also run a script by clicking on it.
