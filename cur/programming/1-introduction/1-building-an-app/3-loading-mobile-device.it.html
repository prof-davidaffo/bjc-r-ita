Condividere il tuo gioco

Il gioco non è ancora finito, ma è abbastanza completato. In questa pagina, condividerai il tuo progetto così che altri possano giocare al tuo gioco.
  1. Puoi usare una delle opzioni seguenti se hai un account Snap!. Se non hai un account Snap!, devi usare l'opzione A.

    Opzione A: Condividi il progetto come file

    Inviare un file XML invia una versione statica del tuo progetto.
    • Nella pagina precedente, hai imparato a salvare un file XML salvandolo sul tuo computer invece che nel cloud. Puoi anche andare al menu File () e selezionare "Export project..." per generare un file XML.
    • Puoi inviare questo file XML ad altre persone. Per aprirlo, possono andare su Snap! e trascinare il file XML. Possono anche andare al menu File () e scegliere "Open..." per aprire il file.
    • Se apporti modifiche al tuo progetto, devi salvarlo di nuovo e inviare il nuovo file perché gli altri vedano gli aggiornamenti.

    Opzione B: Condividi il progetto come link

    Condividendo il tuo progetto si creerà un link che consentirà ad altri di vedere la versione salvata più recente del tuo progetto.

    Creare il link

    1. Assicurati che il progetto sia salvato.
    2. Seleziona "Open..." dal menu File in Snap!, .
    3. Seleziona, dall'elenco dei progetti, il progetto che vuoi condividere (il tuo progetto attuale probabilmente si chiama "U1L1-ClickAlonzo").
      • Per trovare un file, puoi digitare una parola nella barra di ricerca in alto per vedere solo i file che corrispondono (nell'esempio a destra compaiono solo i file che corrispondono a "U1").
    4. Fai clic su "Share", poi su "Yes" per condividere. Infine fai clic su "Cancel" per chiudere il menu.
    5. Ora l'URL (il link) nella barra di navigazione della finestra è cambiato. Copia il nuovo URL e puoi inviarlo via email. Quando hai tempo, puoi inviarlo a qualcun altro per condividere il progetto.
      • Fai clic per vedere le immagini che mostrano come cambia l'URL.

      Se apporti modifiche al progetto dopo averlo condiviso, queste modifiche si riflettono in ciò che vedono le altre persone, quindi non devi condividerlo di nuovo.

  2. Se lavori in gruppo, assicurati che tutti abbiano una copia del progetto.

    Salvare una copia di un progetto

    • Sia che tu condivida il progetto come file o come link, quando il secondo compagno salva il progetto (sul suo computer o nel suo account), sta salvando una copia— non un file condiviso. Se il secondo compagno modifica il progetto, dovrà salvarlo di nuovo e condividerlo (come file o link) con il primo compagno per fargli vedere la modifica.
      Se tu e il tuo compagno avete account Snap!, il progetto verrà salvato nell'account della persona che ha effettuato l'accesso. Per cambiare chi ha effettuato l'accesso in Snap!, apri il menu Cloud () e scegli "Sign out". Lascia che il tuo compagno acceda a Snap!.
