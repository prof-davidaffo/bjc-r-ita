: Scorrere una lista
AAP-2.O.2

Scorrere una lista significa guardare ogni elemento della lista. For each è iterativo. Cioè, è ripetitivo, come for, che può anche scorrere una lista. Ma a differenza di for, for each scorre la lista senza usare numeri di indice.

AAP-2.O.3
La procedura 
goto()
non è incorporata nel linguaggio dell'AP, quindi è scritta in minuscolo come le altre procedure definite dai programmatori.
Lo script verrebbe scritto come 
FOR EACH point IN myDrawing
{
    goto(point)
}
oppure .
Come sarebbe questo in Python? 

for each point in myDrawing:
    goto(point)

  2. PROCEDURE Mystery(numberList, targetNumber)
{
  counter ← 0
  FOR EACH number IN numberList
  {
    IF(number > targetNumber)
    {
      counter ← counter + 1
    }
  }
  RETURN(counter)
}
myMathGrades ← [100, 80, 90, 80, 60, 100, 50, 100]
Mystery(myMathGrades, 60)
