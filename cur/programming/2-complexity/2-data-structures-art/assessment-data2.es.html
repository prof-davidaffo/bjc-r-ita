Quali degli script qui sotto produrranno il seguente comportamento dello sprite?

(Quando esegui lo script, però, in realtà dice Abraham Lincoln una sola volta.)

Scegli tutte le risposte corrette.

Corretto.
No, quando uno sprite dice una lista, vediamo una rappresentazione visiva della lista.
No, questo dice "Abraham Lincoln" tutto insieme, non in due fumetti separati.
Corretto.

inputList ← [3, -1, 2, 10, -5]



FOR EACH item IN inputList

{

  IF(item > 0  AND  item * item > 4)

  {

    DISPLAY(item)

  }

}

2.2.3 
PROCEDURE Mystery(numberList, targetNumber)

{
AAP-

  counter ← 0

  FOR EACH number IN numberList

  {

    IF(number > targetNumber)

    {

      counter ← counter + 1

    }

  }

  RETURN(counter)

}

myMathGrades ← [100, 80, 90, 80, 60, 100, 50, 100]

Mystery(myMathGrades, 60)

Che cosa verrà restituito se il codice sopra viene eseguito?
[100, 80, 90, 80, 100, 100]
Che tipo di valore restituisce la procedura?
7
60 > 60 ?
6
Corretto!
60
Che cosa fa la procedura?
